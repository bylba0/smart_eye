# Smart Eye 👁️ — AI Security System for Raspberry Pi

Интеллектуальная система безопасности на базе компьютерного зрения, способная распознавать нарушителей и реагировать на жесты в реальном времени.

## 🚀 Основные функции
* **AI Detection**: Распознавание скелета человека (Pose Estimation) с помощью YOLOv8.
* **Smart Alert**: Аппаратная тревога (зуммер/светодиоды), которая отключается жестом "Руки вверх".
* **Heatmap Analytics**: Генерация тепловой карты перемещений для анализа "слепых зон".
* **Local Archive**: Автоматическое сохранение фото нарушителей и отчетов в локальный веб-архив.
* **On-Premises**: Полная автономность и приватность данных (работает без облаков).

## 🛠 Технологический стек
- **Hardware**: Raspberry Pi 4/5, USB/IP-Camera, Buzzer, LEDs.
- **Vision**: Python, OpenCV, Ultralytics (YOLOv8).
- **Web**: Python HTTP Server для удаленного доступа к логам.

## ⚙️ Технические особенности и Оптимизация (Performance)

При разработке для Edge-устройств (Raspberry Pi) основной упор был сделан на баланс между точностью и скоростью обработки (Latency):

* **Оптимизация инференса**: Использована модель YOLOv8n-pose с уменьшенным входным разрешением (`imgsz=160`). Это позволило добиться стабильных **6-8 FPS** на CPU Raspberry Pi без использования внешних ускорителей.
* **Управление очередью кадров**: Реализована очистка буфера OpenCV (`cap.grab()`), что минимизирует задержку (lag) видеопотока до <150мс.
* **Логика детекции**: Использованы ключевые точки (Keypoints 11, 12) для построения тепловых карт, что менее ресурсозатратно, чем сегментация или полноценный трекинг объектов.
* **Fail-safe система**: Локальное сохранение данных (Local Archive) работает параллельно с основным циклом детекции, не блокируя поток обработки видео.

## 📊 Масштабируемость
Архитектура проекта позволяет легко заменить модуль вывода (Visualizer) или добавить новые жесты через расширение логики в `core/gestures.py`. Проект готов к интеграции в экосистему ROS2 в качестве отдельного нода.
Для работы системы используется Raspberry Pi 4 и внешние модули сигнализации.

| Компонент | Пин Raspberry Pi (BCM) | Описание |
|-----------|------------------------|----------|
| Buzzer    | GPIO 18                | Звуковая сирена |
| Red LED   | GPIO 23                | Индикатор тревоги |
| Green LED | GPIO 24                | Индикатор "Безопасно" |

### Алгоритм работы:
1. **Режим ожидания**: Горит зеленый LED.
2. **Обнаружение**: Включается красный LED и зуммер.
3. **Авторизация**: При распознавании жеста "Руки вверх" (Hands Up) система переходит в режим ожидания на 10 секунд.
